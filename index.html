The Rive App

The app allows teachers to upload and download past exam papers, letting them to exchange test materials between schools. Previously it was limited to joining WhatsApp groups and exchanging between known contacts.

Papers are organized according to subjects and respective classes. Before downloading the app allows you to preview a paper with an inbuilt PDF reader.

Technologies

Developed in Android Studio. Mostly made in Flutter. Java for Android.
Dev Period: 4weeks (3weeks planning & learning Flutter, 1week coding)
Documents are hosted on Cloud Firestore, a Google NoSQL document database.
Uses NoSQL Google Firebase Realtime Database for data manipulation and realtime synchronization.

Download

The app apk (android) can be downloaded below. It was never launched on iOS since everyone used an Android smartphone. Flutter also allowed releasing the app as a web app but all end-users (teachers) prefered smartphones and never used laptops.

Teacher Mark Book

This is a tool that was designed to replace a manual mark book used in schools. Originally, in Education large books are provided to capture students marks in a year, do analysis in another book and students reports on another. The process was long and took days, used a lot of stationery and prone to errors. Since the Ministry was not ready for an automated Web System an excell tool was the best available choice.

The system capture student marks, calculates performance, produce a result analysis per class, and populates reports for each individual and a continuous assessment form for a learner in a year.
